# ambev-project-thiago
Projeto de automação de testes end-to-end (E2E) desenvolvido com Cypress para validação da interface web de uma aplicação baseada em cenários inspirados em processos da Ambev. O projeto segue boas práticas de organização, reutilização de comandos, Page Objects e geração de evidências para facilitar a manutenção e escalabilidade da suíte de testes.
